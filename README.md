# Task 02 - Exploratory Data Analysis

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the cleaned sales dataset. The goal was to understand the dataset, identify patterns, analyze product sales, study customer behavior, and detect unusual transactions through statistical analysis.

## Tasks Performed

* Examined the dataset structure and dimensions.
* Displayed column names and dataset information.
* Calculated descriptive statistics for numerical columns.
* Analyzed total sales for each product.
* Examined payment method usage.
* Studied order status distribution.
* Analyzed referral source distribution.
* Detected outliers in the TotalPrice column using the IQR method.
* Identified and displayed outlier records.

## Tools Used

* Python
* Pandas
* VS Code

## Outcome

The analysis provided valuable insights into product performance, customer purchasing patterns, payment preferences, and outlier transactions. The findings can help support better business decisions and further data analysis.
